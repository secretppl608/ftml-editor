### 这是什么？
这是一个维基代码编辑器，基础功能包括拖拽组件生成各个维基平台语法，通过相当轻量级别的操作可以快速可视化生成维基代码并导出，可以预览CSS效果，可以管理作品计划、未来计划开发高级功能，例如多人协作
请访问[官方网站](https://secretppl608.github.io/ftml-editor)获取exe可执行文件，预期会制造移动端适配版本。
